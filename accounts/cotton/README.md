now # Django-Cotton Components for Accounts App

This directory contains reusable UI components for the accounts app, following Django-Cotton conventions.

## Structure

Components for authentication and user-related UI will be created here as needed.

## Usage

Components are used in templates using Django-Cotton syntax:

```html
<c-accounts.login-button />
```

## Naming Convention

- Use kebab-case for component files
- Component names map to template paths
